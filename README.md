# MacroRecorder v1.0.0 ✨ 初始版本

一个功能强大的 Windows 桌面宏录制与播放工具，支持鼠标和键盘操作录制，自定义播放速度和循环执行。

---

## 🎯 核心功能
- **全局录制**：捕获鼠标移动、点击、滚轮和键盘按键
- **精确回放**：高精度时间控制，准确重现操作
- **循环播放**：可设置无限循环执行宏操作
- **托盘常驻**：后台运行不干扰工作
- **全局热键**：⚠️ 注册或修改全局热键可能需要管理员权限
- **自定义快捷键**：支持 Ctrl / Alt / Shift 组合
- **宏文件管理**：保存 / 加载 JSON 格式宏文件

---

## 🚀 安装说明
1. 下载最新 Release 版本 `MRsetup.exe`
2. 双击安装并运行
3. 注意：GitHub 自动生成的 **Source code (zip / tar.gz)** 压缩包无需下载，里面只有仓库快照或 README

---

## 🖥 系统要求
- Windows 10 / 11
- .NET Desktop Runtime 10.0+ 或更高版本
- 管理员权限推荐（修改全局热键时需要）

---

## 📖 使用指南
### 快速操作
- **开始录制**：F9
- **停止录制**：F10
- **播放宏**：F7
- **停止播放**：F8

> 💡 提示：快捷键可自定义，点击主窗口“快捷键设置”进行修改

### 播放速度
- 0.5x：慢速，精确操作
- 1x：原速
- 2x：快速重复操作

### 循环播放
- 勾选循环播放，宏将持续执行直到手动停止
