# MacroRecorder ✨

一个功能强大的 Windows 桌面宏录制与播放工具，支持鼠标键盘操作的高精度捕获、自定义回放及“非破坏性”预览功能。

---

## 🎯 核心功能
- **全局录制/暂停**：捕获鼠标移动、点击、滚轮和键盘按键。支持录制中途暂停，排除干扰。
- **非破坏性预览**：⚠️ **特色功能**。录制暂停时可随时播放检查，结束后自动回到录制状态，无需重录。
- **精确回放**：高精度时间控制，支持 **0.5x / 1.0x / 2.0x** 速度调节。
- **循环/定时播放**：支持无限循环执行及倒计时启动。
- **全局热键**：支持自定义快捷键，最小化到托盘时依然精准响应。
- **托盘常驻**：现代化 UI 菜单，录制时图标动态提醒，不占用任务栏。
- **宏文件管理**：支持将操作序列保存为 JSON 格式，方便随时加载。

---

## 🚀 安装与运行
1. 前往 [Latest Release](https://github.com/p1aychess/MacroRecorder-Official/releases/latest) 下载最新的 `MRsetup.exe`。
2. 双击安装并运行。
3. **注意**：建议以 **管理员权限** 运行，以确保全局热键和模拟操作在所有窗口下生效。

---

## 🖥 系统要求
- Windows 10 / 11
- .NET Desktop Runtime 10.0+ 或更高版本

---

## 📖 使用指南

### 快捷键操作 (默认)
| 功能 | 快捷键 | 说明 |
| :--- | :--- | :--- |
| **录制 / 暂停** | `F9` | 切换录制与暂停状态 |
| **停止录制** | `F10` | 结束录制并生成脚本 |
| **播放 / 暂停** | `F7` | 切换播放与暂停；录制暂停时点击可预览 |
| **停止播放** | `F8` | 立即终止宏执行 |

> 💡 **提示**：所有快捷键均可在主窗口的“快捷键设置”中进行自定义修改。
