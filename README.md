# MacroRecorder v2.0.0 
✨ 生产力大幅提升版

一个功能强大的 Windows 桌面宏录制与播放工具。v2.0.0 引入了**“非破坏性预览”**技术，让宏制作像视频剪辑一样简单灵活。

---

## 🌟 2.0 版本新特性
- **非破坏性预览**：在录制暂停时可随时播放检查，播放完自动回到暂停状态，无需中断录制流程。
- **全场景暂停/继续**：录制和播放过程中均支持一键暂停（F9/F7），精准控制每一个环节。
- **智能钩子管理**：播放期间自动隔离录制钩子，防止操作回环。
- **视觉反馈增强**：录制时托盘图标红色闪烁，界面状态色随工作模式智能切换。

---

## 🎯 核心功能
- **全局录制**：捕获鼠标移动、点击、滚轮和键盘按键。
- **精确回放**：高精度时间控制，准确重现操作。
- **循环播放**：支持无限循环执行，适合自动化重复任务。
- **定时执行**：支持设置倒计时开始，预留准备时间。
- **托盘常驻**：现代化圆角菜单，后台运行更丝滑。
- **自定义快捷键**：支持全局热键修改，适配不同用户习惯。

## 📖 使用指南

### 快速操作表
| 功能 | 默认热键 | 说明 |
| :--- | :--- | :--- |
| **录制 / 暂停录制** | `F9` | 暂停时可进行预览或休息，不记录多余操作 |
| **停止录制** | `F10` | 彻底结束本次录制 |
| **播放 / 暂停播放** | `F7` | 播放时可暂停观察；**录制暂停时**按此键可进行预览 |
| **停止播放** | `F8` | 强行终止当前宏执行 |

> 💡 **小技巧**：在录制长脚本时，先按 `F9` 暂停，再按 `F7` 预览刚才录的对不对，如果没问题，再按 `F9` 接着录！
